# Muwanx Examples

このディレクトリには、Muwanxビューアーの使用例が含まれています。

## 使い方

メインアプリケーションは `src/viewer/` にあり、`npm run dev` で起動します。

## カスタム例の追加

将来的に、このディレクトリにカスタムシーン、設定ファイル、ONNXモデルなどを追加できます。

例：
```
examples/
  └── custom-robot/
      ├── scene.xml
      ├── config.json
      └── policy.onnx
```

詳細は `docs/ADDING_CUSTOM_CONTENT.md` を参照してください。
